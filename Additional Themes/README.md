## Additional Themes

#### Installation

1. Copy to `/EFI/OC/Resources/Image`
2. Open `config.plist`
3. Go to `PickerVariant`
4. To change a theme, enter NameOfArtsit\NameOfTheme.</br>
	**Examples**: `Antuneddu\Blue` or `chris1111/Minimal-SSD`
5. Save, reboot.
